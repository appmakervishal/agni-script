Agni-Script

Unlock Lua, PHP, and Python support in your web pages!

How to Unlock Additional Languages

To enable Lua, PHP, or Python in your HTML pages using Agni-Script, include the following scripts:

<script src="https://appmakervishal.github.io/agni-script/lua.js"></script> <script src="https://appmakervishal.github.io/agni-script/php.js"></script> <script src="https://appmakervishal.github.io/agni-script/py.js"></script>

Use the Language Tags

Once the scripts are included, you can write code directly in your HTML using these custom tags:

<lua>…Lua code…</lua> <php>…PHP code…</php> <py-script>…Python code…</py-script>

Example Usage

Your HTML page could look like this:

<!DOCTYPE html>
<html>
<head>
  <script src="https://appmakervishal.github.io/agni-script/lua.js"></script>
  <script src="https://appmakervishal.github.io/agni-script/php.js"></script>
  <script src="https://appmakervishal.github.io/agni-script/py.js"></script>
</head>
<body>
  <lua>
    print("Hello Lua!")
  </lua>
  <php>
    echo "Hello PHP!";
  </php>
  <py-script>
    print("Hello Python!")
  </py-script>
</body>
</html>

Notes

  • This page is informational — it execute your code in client-side
  • Including the scripts on your own web page enables Agni-Script to recognize and run these tags.
  • Make sure your users have access to the internet to load the scripts from GitHub Pages.
